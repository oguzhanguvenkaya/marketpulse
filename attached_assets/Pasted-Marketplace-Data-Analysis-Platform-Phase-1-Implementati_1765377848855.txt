Marketplace Data Analysis Platform - Phase 1 Implementation Plan
This plan outlines the steps to build the MVP (Phase 1) of the Marketplace Data Analysis Platform, focusing on Hepsiburada integration, keyword analysis, and a basic dashboard.

Proposed Changes
Project Structure & Infrastructure
We will set up a monorepo-style structure or separate folders for backend and frontend.

[NEW] 
docker-compose.yml
PostgreSQL service (Persistent volume)
Redis service
(Optional) Adminer or generic DB viewer
[NEW] 
backend/
Dependencies: fastapi, uvicorn, sqlalchemy, alembic, psycopg2-binary, celery, redis, playwright, beautifulsoup4, openai.
Structure:
app/main.py: Entry point.
app/db/: Database session and models (products, product_snapshots).
app/api/: API Routes (search, products, tasks).
app/services/: Business logic (scraping, llm, analysis).
app/worker.py: Celery app configuration.
Backend Development
[NEW] 
backend/app/db/models.py
Product model (id, platform, external_id, name, url, etc.)
ProductSnapshot model (price, rating, reviews, stock status, snapshot_date)
[NEW] 
backend/app/services/scraping.py
ScrapingService class using playwright (async).
Integration with Bright Data proxy.
Hepsiburada-specific parsing logic.
[NEW] 
backend/app/tasks.py
Celery tasks for background scraping.
Redis locking to prevent duplicate scraping of the same product on the same day.
Frontend Development
[NEW] 
frontend/

Pages:
Dashboard: Overview charts.
Search: Input for keyword search and list of results.
ProductDetail: Historical price/rating charts for a specific product.
Verification Plan
Automated Tests
Backend Unit Tests: pytest for API endpoints and database models.
Scraping Tests: Mocked HTML tests to verify parsing logic without hitting live sites.
Manual Verification
Infrastructure: Run docker-compose up and verify DB/Redis connection.
API: Use Swagger UI (/docs) to trigger a search task and check status.
End-to-End:
Open Frontend.
Search for a keyword (e.g., "iphone 13").
Verify "Task Started" notification.
Wait for completion.
Verify results appear in the list.
Click a product and verify details/charts.
