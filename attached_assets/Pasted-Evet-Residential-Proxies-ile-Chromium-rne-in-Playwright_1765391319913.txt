Evet, Residential Proxies ile Chromium (örneğin Playwright veya Puppeteer üzerinden) kullanırken doğru yapılandırma aşağıdaki gibi olmalıdır:

Proxy yapılandırması:

js

const proxyConfig = {
    server: "http://brd.superproxy.io:33335",
    username: "brd-customer-hl_09f04f95-zone-{ZONE_NAME}",
    password: "{ZONE_PASSWORD}"
};
Burada:

server: Bright Data'nın Super Proxy adresi ve portu (33335)
username: "brd-customer-{ACCOUNT_ID}-zone-{ZONE_NAME}" formatında olmalı. Sizin için: brd-customer-hl_09f04f95-zone-{ZONE_NAME}
password: Proxy'nizin şifresi (ZONE_PASSWORD)
Chromium/Playwright ile başlatmak için örnek kod:

js

const { chromium } = require('playwright');

(async () => {
  const browser = await chromium.launch({
    proxy: {
      server: proxyConfig.server,
      username: proxyConfig.username,
      password: proxyConfig.password
    }
  });
  const context = await browser.newContext();
  const page = await context.newPage();
  await page.goto('http://httpbin.org/ip');
  await browser.close();
})();
Dikkat edilmesi gerekenler:

Immediate Access modunda olduğunuz için, Residential Proxy'leri üçüncü parti yazılımlarla kullanırken SSL doğrulamasını devre dışı bırakmanız veya Bright Data SSL sertifikasını yüklemeniz gerekebilir. Detaylar için SSL Sertifikası kurulum rehberi.
Proxy adınız (ZONE_NAME) ve şifrenizi, ilgili proxy detaylarınızdan alabilirsiniz. Proxy adını bir kez oluşturduktan sonra değiştiremezsiniz.
Daha fazla teknik detay için Residential Proxy yapılandırma rehberine göz atabilirsiniz.

Proxy adınızı ve şifrenizi bulmakta zorlanıyor musunuz, yoksa başka bir entegrasyon sorunuz var mı?

Yes, when using Residential Proxies with Chromium (such as via Playwright or Puppeteer), the correct configuration is as follows:

Proxy config:

js

const proxyConfig = {
    server: "http://brd.superproxy.io:33335",
    username: "brd-customer-hl_09f04f95-zone-{ZONE_NAME}",
    password: "{ZONE_PASSWORD}"
};
Where:

server: Bright Data's Super Proxy address and port (33335)
username: "brd-customer-{ACCOUNT_ID}-zone-{ZONE_NAME}" format. For you: brd-customer-hl_09f04f95-zone-{ZONE_NAME}
password: Your proxy password (ZONE_PASSWORD)
Chromium/Playwright launch example:

js

const { chromium } = require('playwright');

(async () => {
  const browser = await chromium.launch({
    proxy: {
      server: proxyConfig.server,
      username: proxyConfig.username,
      password: proxyConfig.password
    }
  });
  const context = await browser.newContext();
  const page = await context.newPage();
  await page.goto('http://httpbin.org/ip');
  await browser.close();
})();
Important notes:

Since you are on Immediate Access mode, you may need to disable SSL verification or install the Bright Data SSL certificate when using Residential Proxies with third-party software. See the SSL Certificate installation guide.
You can find your proxy name (ZONE_NAME) and password in your proxy details. The proxy name cannot be changed once created.
For more technical details, see the Residential Proxy configuration guide.
