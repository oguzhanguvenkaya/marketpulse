## 1. 📊 Yönetici Özeti ve Genel Değerlendirme

Genel durum: Proje fonksiyonel kapsam olarak güçlü ve gerçek ürün ihtiyaçlarını karşılayacak şekilde gelişmiş. Ancak mevcut haliyle canlı ortam için kritik güvenlik açıkları ve ölçekleme riskleri barındırıyor.

### 👍 İyi Taraflar (Pros)

1. Servis katmanı ayrımı mevcut: scraping, price monitor, transcript gibi iş kuralları ayrı servislerde.

1. Güvenlik modelinde kritik açıklar var: path traversal riski, frontend secret exposure, TLS doğrulama devre dışı.

4. Çok sayıda broad exception/silent fail (`except: pass`) var.
5. Performans tarafında N+1 ve full-scan path'ler mevcut.

## 7. 🚀 Aksiyon Planı ve Refactoring Önerileri

### Öncelikli 3-5 Adım (Sıralı)

1. **P0 - Secret modelini düzelt:** `VITE_INTERNAL_API_KEY` kullanımını kaldır, backend auth'ı client-visible secret olmadan yeniden tasarla.
2. **P0 - Path traversal kapat:** SPA static serve için canonical path doğrulaması ekle.
3. **P1 - TLS doğrulamasını aç:** `ssl=False`, `CERT_NONE`, `verify=False` kullanımını kaldır.
4. **P1 - Runtime bug fix + regression test:** `SellerSnapshot.fetched_at` kullanımını `snapshot_date` ile düzelt.
5. **P1 - Performans refactor:** N+1 sorguları toplu SQL sorgularına taşı, alert filtre yolunu SQL-level paginated hale getir.

---

### Refactoring Örneği 1: Path Traversal

**Mevcut Kod (Before)**  
Dosya: `backend/app/main.py`

```python
file_path = os.path.join(frontend_dist, full_path)
if full_path and os.path.isfile(file_path):
    return FileResponse(file_path)
```

**Olması Gereken Kod (After)**

```python
from pathlib import Path

frontend_root = Path(frontend_dist).resolve()
requested = (frontend_root / full_path).resolve()

if full_path and requested.is_file() and frontend_root in requested.parents:
    return FileResponse(str(requested))
```

### Refactoring Örneği 2: Yanlış Alan Adı Kaynaklı Runtime Hatası

**Mevcut Kod (Before)**  
Dosya: `backend/app/api/store_product_routes.py`

```python
latest_snapshot = db.query(SellerSnapshot).filter(
    SellerSnapshot.monitored_product_id == mp.id,
    SellerSnapshot.price.isnot(None)
).order_by(SellerSnapshot.fetched_at.desc()).first()
```

**Olması Gereken Kod (After)**

```python
latest_snapshot = (
    db.query(SellerSnapshot)
    .filter(
        SellerSnapshot.monitored_product_id == mp.id,
        SellerSnapshot.price.isnot(None),
    )
    .order_by(SellerSnapshot.snapshot_date.desc())
    .first()
)
```
