Extraction Method:
Parse HTML source
Find <script> tag containing window["__envoy_sexual-content-warning-popup__PROPS"]
Extract JSON string using regex: window\["__envoy.*?__PROPS"\]\s*=\s*(\{.*?\});
Parse JSON and navigate to product.merchantListing
Extract merchant data from structured JSON objects
Note: This is Server-Side Rendered (SSR) data, meaning the JSON is pre-loaded in the HTML source. No JavaScript execution is required to access this data - it's available in the raw page source (View Source / Ctrl+U).
